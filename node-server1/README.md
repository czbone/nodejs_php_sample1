# Express.js and EJS

Example of using Node.js, Express.js and EJS.

### 動作環境

Vagrant環境で動作します。
https://github.com/czbone/lemp_nodejs_centos7

/var/www/node-appに配置します。

### Endpoints

+ `/`
+ `/student`
+ `/teacher`

## Install

`npm install`

## Run

1. `npm start`
2. Open `http://[Server IP]` in a web browser.
